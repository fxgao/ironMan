帮助信息

欢迎使用华三网站宝！

本页面旨在提供最方便全面的常用网站聚合，提供导航便利，轻松工作。

您可以将html文件的快捷方式放在桌面上，或者把这个地址加到浏览器的收藏夹。
设置为浏览器首页更方便^_^

  • 如何添加自定义网址

    • 在文件根目录下打开bookMarks.js,可以通过编辑器对该文件进行编辑。只需要在数组中添加如下格式的数据，即可添加一条新的网址到你的页面上。

                        ,(逗号记得加哦)
                    {
                        name: "H3C官网",    
                        Url: "http://www.h3c.com",  
                        image: "statics/1.jpg", 
                        type: "其他网站",   
                        weight: 0   
                    }

                    属性解释：
                    name// 网站的名字
                    Url// 网站的链接
                    image// （可选）图片链接，可以不填写，会用默认图片替代。
                    type// （可选）网站分类（不写该项默认分类到“其他网站”）
                    weight// 权重，代表该网站的使用频率。
